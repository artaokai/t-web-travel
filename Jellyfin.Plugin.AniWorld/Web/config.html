AniWorld Downloader Settings

General

Maximum number of simultaneous downloads (1-5). Lower values are recommended to avoid rate limiting.
Number of times to retry a failed download (0 = no retries). Uses exponential backoff between attempts and will try alternative providers on failure.
Automatically trigger a Jellyfin library scan when a download completes. This makes new episodes appear in your library immediately.
Allow non-admin users to access the AniWorld Downloader UI via the sidebar. Requires the File Transformation plugin (recommended) or falls back to direct index.html modification. A server restart is required after changing this setting.

AniWorld Downloader (aniworld.to)

Enable searching and downloading anime from aniworld.to.
Base directory for downloaded anime files. Files will be organized as: DownloadPath/SeriesName/Season XX/SeriesName - SXXEXX.mkv
Default language for AniWorld downloads.
Default streaming provider for AniWorld. Vidmoly offers superior download speeds when available. VOE is a good fallback due to its wide availability.
If all retries with the preferred provider fail, the download will be retried with this provider.

HiAnime Downloader (hianime.to)

Enable searching and downloading anime from hianime.to (English sub/dub).
Base directory for downloaded anime files. Leave empty to use the AniWorld download path.
Default language for HiAnime downloads.

s.to Downloader (SerienStream)

Enable searching and downloading TV series from s.to.
Base directory for downloaded series files. Leave empty to use the AniWorld download path.
Default language for s.to downloads.
Default streaming provider for s.to.
If all retries with the preferred provider fail, the download will be retried with this provider.
